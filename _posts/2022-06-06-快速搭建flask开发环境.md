<h3>如何快速搭建一个flask开发环境</h3>
<p>1.在使用的文件夹中创建虚拟环境</p>
 <p>2.安装虚拟环境首先需要安装virtualenv模块</p>
   <p>3.创建虚拟环境,virtualenv flask_env</p>
   <p>4.重新打开命令行</p>
   <p>5.安装flask模块----pip install flask</p>
   
   
    ~~~
    
        from flask import Flask
        
        # 创建对象
        app = Flask(__name__)

        # 路由的地址
        @app.route('/')
        def index():
            # return 代表将数据返回给浏览器
            return "Hello Flask"


        if __name__ == '__main__':
            # 启动web应用
            app.run(debug=true)
    ~~~
