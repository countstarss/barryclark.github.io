<p>1.在使用的文件夹中创建虚拟环境</p>
<p>2.安装虚拟环境，首先需要安装virtualenv模块</p>
 
    pip install virtualenv
    
<p>3.创建虚拟环境</p>
 
    virtualenv flask_env
    
<p>4.重新打开命令行</p>
<p>5.安装flask模块</p>

    pip install flask
